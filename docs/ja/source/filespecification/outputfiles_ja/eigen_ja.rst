.. _subsec:eigen.dat:

eigen.dat
~~~~~~~~~~

収束したハミルトニアンの固有値が低エネルギー順に出力されます。
以下にファイル例を記載します。

::

    0, -1.4330804705460427
    1, -1.4330775519300982
    2, 0.32326944941464825
    3, 0.3232694494155258
    4, 0.323269449420004
    5, 0.32326944942497615
    6, 0.3232694494449527
    ...

ファイル形式
^^^^^^^^^^^^

:math:`[`\ int01\ :math:`]` :math:`[`\ double01\ :math:`]`

パラメータ
^^^^^^^^^^

-  :math:`[`\ int01\ :math:`]`

   **形式 :** int型

   **説明 :**
   ハミルトニアンの固有値のindex。

-  :math:`[`\ double01\ :math:`]`

   **形式 :** double型

   **説明 :**    ハミルトニアンの固有値。
